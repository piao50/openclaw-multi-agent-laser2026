# 第1章 引言与技术背景

## 1.1 脉冲光纤激光器概述

脉冲光纤激光器是以掺杂光纤为增益介质，通过调Q、锁模或外调制等方式产生脉冲输出的一类激光器。与传统固体脉冲激光器（如Nd:YAG、Ti:Sapphire）相比，其核心优势在于：

- **散热效率高**：纤芯直径小（~10 μm）、表面积/体积比大，热管理天然优于块状晶体
- **光束质量优**：单模光纤输出接近衍射极限（M² ≈ 1.0–1.1）
- **结构紧凑**：全光纤化设计可实现手持或桌面级系统
- **可靠性高**：无对准敏感的光学元件，抗震动、免维护
- **成本递减**：规模化生产推动单位成本持续下降

## 1.2 发展历程

| 时期 | 里程碑 |
|------|--------|
| 1990s | 第一台掺铒光纤激光器诞生；调Q光纤激光器实现 |
| 2000s | SESAM 锁模光纤激光器实现飞秒脉冲；IPG 推出首台商用高功率连续光纤激光器 |
| 2010s | 皮秒/飞秒光纤 MOPA 系统商业化；超快激光精密加工兴起 |
| 2020–2024 | kW 级纳秒脉冲输出；空芯光纤脉冲压缩突破；2 μm 脉冲光纤激光器成熟 |
| 2025–2026 | 多芯光纤相干组束、反共振空芯光纤、AI 驱动脉冲优化等新方向涌现 |

## 1.3 技术分类

### 按脉冲宽度分类

```
脉冲宽度谱系：

纳秒 (ns)    1–500 ns    → 宏加工（切割、焊接、标记）
皮秒 (ps)    0.5–50 ps   → 精密微加工（划片、打孔）
飞秒 (fs)    10–500 fs   → 超精密加工、阿秒驱动
阿秒 (as)    < 1 ps      → 基础科学（电子动力学）
```

### 按增益介质分类

| 增益离子 | 波长范围 | 主要应用 |
|----------|---------|---------|
| Yb³⁺ | 1030–1080 nm | 工业加工、科研 |
| Er³⁺ / Er³⁺:Yb³⁺ | 1550–1565 nm | LiDAR、通信、eye-safe |
| Tm³⁺ | 1900–2050 nm | 医疗、气体检测 |
| Ho³⁺ | 2050–2150 nm | 医疗、国防 |
| Pr³⁺ | 635 nm, 605 nm | 可见光应用 |
| Nd³⁺ | 1060 nm, 920 nm | 早期光纤激光 |

### 按脉冲产生机制分类

1. **调Q（Q-switching）**
   - 声光调Q（AO Q-switch）：主流，重复频率 1–500 kHz
   - 电光调Q（EO Q-switch）：高速，适合 ns 级精确控制
   - 可饱和吸收体被动调Q：结构简单，适合低功率场景

2. **锁模（Mode-locking）**
   - 被动锁模：SESAM、NPRL、2D 材料 SA
   - 主动锁模：幅度/相位调制器驱动
   - 混合锁模：主动+被动结合

3. **种子源 MOPA**
   - 半导体激光器种子 + 多级光纤放大
   - 脉冲参数（宽度、频率、能量）独立可调
   - 工业主流方案
